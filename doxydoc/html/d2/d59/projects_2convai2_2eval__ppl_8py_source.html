6 """
7 Base script for running official ConvAI2 validation eval for perplexity. This uses a the
8 version of the dataset which does not contain candidates. Leaderboard scores will be run
9 in the same form but on a hidden test set.
10 
11 The official vocabulary for the competition is based on using the
12 "split_tokenize" method on in the ParlAI core dictionary (parlai/core/dict.py)
13 and is built on the training and validation sets of the "convai2" task.
14 This dictionary contains a total of 19304 tokens. The test set contains some
15 tokens which are not in this dictionary--this tokens will not be provided, but
16 we will also *SKIP* calculating perplexity on these tokens. The model should
17 still produce a good guess for the remaining tokens in the sentence, so
18 handling unknown words or expanding the vocabulary with pre-trained or
19 multitasked embeddings are legitimate strategies that may or may not impact the
20 score of the models.
21 
22 Note that this tokenizer will also be used during the perplexity evaluation:
23 the model will be asked to predict one word at a time according to this
24 tokenizer's parsing of the text.
25 
26 This requires agents to implement the following function:
27 
28 def next_word_probability(self, partial_out):
29  Return probability distribution over next words given a partial true output.
30  This is used to calculate the per-word perplexity.
31 
32  Arguments:
33  partial_out -- list of previous "true" words
34 
35  Returns a dict, where each key is a word and each value is a probability
36  score for that word. Unset keys assume a probability of zero.
37 
38  e.g.
39  {'text': 'Run test program.'}, ['hello'] => {'world': 1.0}
40 """

59 class WordFrequencyEntry(Agent):
60  """
61  This is an example entry which tries to use the RepeatLabelAgent. Since no labels
62  are given to the model, it will guess something useless.
63 
64  It builds the official dictionary first, so that it can provide a minimum probablity
65  for each word as well as use the official tokenizer.
66  """

68  def __init__(self, opt, shared=None):
69  super().__init__(opt, shared)
70  if not shared:
71  # build official eval dictionary
72  self.dict = build_dict()
73  else:

77  # set probability of each word, skipping the invalid words like __NULL__

79  self.freqs = {k: f for k, f in self.dict.freqs().items() if f <= max_freq}
