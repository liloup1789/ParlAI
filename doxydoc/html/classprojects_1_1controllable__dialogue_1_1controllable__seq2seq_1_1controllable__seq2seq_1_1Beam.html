    Set to 0 as usual in ParlAI

    Set to 1 as usual in ParlAI
:param eos_token:
    Set to 2 as usual in ParlAI
:param min_n_best:
    Beam will not be done unless this amount of finished hypothesis
    (with EOS) is done
:param cuda:
    What device to use for computations
